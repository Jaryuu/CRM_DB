Proyecto #2 - CRM Empresarial

Requerimientos:
- Java JRE 1.6+
- PostgreSQL 9.4.1
- MongoDB 3.0.3
- Conexión a internet (Requerida para obtener tweets).

Configuración inicial:
1. En PostgreSQL crear la base de datos CRM.
2. Correr el script proyecto2.sql dentro de CRM.


Ejecución:
1. Ejecutar mongod desde la línea de comandos. Esto es requerido para que se puedan manejar las requests de mongo dentro del CRM.
2. Ejecutar el archivo CRM.jar
3. Se puede utilizar ya el CRM, agregando, modificando o quitando clientes.
